Generate a Password

Instructions

  • Click Generate Password to start .
  • When prompted choose the length of the password. Length could be between 8 and 128 characters .
  • When prompted select if you want lowercase in your password string.
  • When prompted select if you want uppercase in your password string.
  • When prompted select if you want numbers in your password string.
  • When prompted select if you want special characters in your password string.
  • When all prompts have been answered a random password will be generated using criteria selected
  • If none of the character types are selected Password will not be generated and user will have to start again.
